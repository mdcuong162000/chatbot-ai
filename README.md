# 🚀 Mika Intelligence — Omnichannel AI Sales System

Mika Intelligence là hệ thống trợ lý bán hàng AI đa kênh, được tối ưu hóa cho thị trường Thái Lan và Việt Nam. Hệ thống mang lại trải nghiệm khách hàng tự nhiên, chuyên nghiệp và giúp tối ưu hóa tỷ lệ chuyển đổi.

---

## ✨ Key Features
- **Omnichannel Support**: Tích hợp Messenger, Zalo, và Web Chat.
- **RAG-Lite Knowledge Base**: Hệ thống nạp kiến thức linh hoạt từ CSV/Excel vào Database.
- **24/7 Automated Sales**: Tự động tư vấn, xử lý từ chối và hỗ trợ chốt đơn hiệu quả.
- **Strategic Reporting**: Báo cáo hiệu suất bán hàng tự động bằng AI.

---

## 🛠️ Technology Stack
- **Backend**: Node.js / Express
- **Database**: SQLite (better-sqlite3)
- **AI Engine**: Llama 3.3 (Groq), GPT-4o (OpenAI)

---

## 🚀 Getting Started

### 1. Cài đặt môi trường
Tạo file `.env` và cấu hình các API Key cần thiết.

### 2. Nạp kiến thức (FAQs)
Sử dụng file mẫu tại `uploads/faq_template.csv` và chạy script đồng bộ dữ liệu.

### 3. Khởi động hệ thống
```bash
npm install
npm start
```
