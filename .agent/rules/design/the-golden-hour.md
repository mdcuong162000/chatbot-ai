# Design System: The Golden Hour Editorial (v3.1)

## 1. Visual Philosophy
- **Concept**: High-end magazine (Editorial) meets Marketing OS.
- **Key Emotion**: Warm, professional, trustworthy, premium.
- **Rule**: No borders (No-Line Rule). Use Surface Hierarchy for separation.

## 2. Color Palette
### Surfaces (Dark Mode)
- Base: `#131313`
- Low: `#1B1B1C`
- High: `#2A2A2A`
- Highest (Modal): `#353535`

### Surfaces (Light Mode / Daylight Hour)
- Base: `#F5F0E8`
- Low: `#FDFAF5`
- High: `#FFFFFF`

### Accents
- **Primary (Amber)**: `#FFD79B`
- **Secondary (Sunset)**: `#FFB59F`

## 3. Typography
- **Headlines**: Plus Jakarta Sans (Tracking -0.02em)
- **Body/Data**: Inter

## 4. Components
- **Buttons**: Metallic Glow (Amber → Gold gradient).
- **Glassmorphism**: 60% opacity + 20px blur (fallback for weak devices).
- **Cards**: xl (1.5rem) border radius for main containers.
- **No Dividers**: Use spacing (16px/20px) instead of `<hr>`.
